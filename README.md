# Breast Cancer Prediction

## Overview
This project uses Machine Learning techniques to predict breast cancer outcomes based on clinical data. The project includes data analysis, model development, visualization, and an interactive Power BI dashboard.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Power BI

## Files
- Breast Cancer Prediction.ipynb
- Breast Cancer.csv
- Dashboard.pbix
- basu ppt final.pptx

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Machine Learning Model Development
- Prediction and Evaluation
- Power BI Dashboard Visualization
